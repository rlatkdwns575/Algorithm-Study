# 1197번 최소 스패닝 트리 (Gold 4)

## 문제 설명
그래프가 주어졌을 때, 그 그래프의 **최소 스패닝 트리(MST, Minimum Spanning Tree)** 를 구하는 프로그램을 작성하는 문제입니다.

최소 스패닝 트리는 주어진 그래프의 모든 정점을 연결하는 부분 그래프 중에서,  
**간선의 가중치 합이 최소인 트리**를 의미합니다.

---

## 입력
- 첫째 줄에 정점의 개수 `V`와 간선의 개수 `E`가 주어집니다.
- 다음 `E`개의 줄에는 각 간선 정보 `A B C`가 주어집니다.
  - `A`번 정점과 `B`번 정점이
  - 가중치 `C`인 간선으로 연결되어 있다는 뜻입니다.

---

## 출력
- 첫째 줄에 **최소 스패닝 트리의 가중치 합**을 출력합니다.

---

## 제한
- `1 ≤ V ≤ 10,000`
- `1 ≤ E ≤ 100,000`
- `C`는 음수일 수도 있으며, 절댓값은 `1,000,000`을 넘지 않습니다.
- 그래프의 정점은 `1`번부터 `V`번까지 번호가 매겨져 있습니다.
- 임의의 두 정점 사이에 경로가 존재합니다.  
  → 즉, **항상 연결 그래프**입니다.
- 최소 스패닝 트리의 가중치 합은
  `-2,147,483,648` 이상 `2,147,483,647` 이하입니다.

---

## 풀이 아이디어

이 문제는 대표적인 **MST(최소 스패닝 트리)** 문제입니다.

### 1. 왜 Kruskal 알고리즘인가?
- 모든 정점을 연결해야 합니다.
- 간선의 가중치 합이 최소가 되어야 합니다.
- 간선의 개수가 최대 `100,000`개이므로,  
  간선 정렬 기반의 `O(E log E)` 풀이가 충분히 가능합니다.

따라서 **Kruskal 알고리즘**을 사용하면 해결할 수 있습니다.

---

## Kruskal 알고리즘이란?
Kruskal 알고리즘은 **가중치가 작은 간선부터 차례대로 선택**하면서,  
**사이클이 생기지 않도록** 간선을 골라 최소 스패닝 트리를 만드는 탐욕적 알고리즘입니다.

### 진행 과정
1. 모든 간선을 **가중치 기준 오름차순 정렬**
2. 작은 간선부터 확인
3. 현재 선택한 간선이 **사이클을 만들지 않으면 채택**
4. 모든 정점이 연결될 때까지 반복

---

## 핵심 자료구조: Union-Find
사이클 여부를 빠르게 확인하기 위해 **Union-Find(Disjoint Set Union)** 자료구조를 사용합니다.

- `find(x)` : x가 속한 집합의 대표를 찾음
- `union(a, b)` : 두 집합을 하나로 합침

두 정점의 대표가 같다면 이미 연결된 상태이므로,  
그 간선을 추가하면 **사이클이 발생**합니다.

---

## 풀이 과정 정리
- 입력받은 간선을 가중치 기준으로 정렬합니다.
- 각 정점의 부모를 자기 자신으로 초기화합니다.
- 정렬된 간선을 순서대로 보면서:
  - 두 정점이 서로 다른 집합이면
    - 간선을 선택하고
    - 두 집합을 합치고
    - 가중치를 누적합니다.
- 최종 누적값이 최소 스패닝 트리의 가중치 합이 됩니다.

---

## 코드

```python
"""
1197번 최소 스패닝 트리 (골드 4)
input : 
    v 정점의 개수, e 간선의 개수
    a, b, c 간선의 연결점과 가중치
output :
    w 스패닝 트리의 가중치
"""
import sys
input = sys.stdin.readline
sys.setrecursionlimit(10**6)

v, e = map(int, input().split())
edge = [tuple(map(int, input().split())) for _ in range(e)]

edge.sort(key=lambda x: x[-1])
group = [i for i in range(v + 1)]

def find(num):
    if num != group[num]:
        group[num] = find(group[num])
    return group[num]

def union(a, b):
    a = find(a)
    b = find(b)
    
    if a != b:
        group[b] = a

length = 0
for a, b, c in edge:
    if find(a) != find(b):
        union(a, b)
        length += c

print(length)
```
