# 9로 나눈 나머지

## 문제 설명

음이 아닌 정수가 문자열 `number`로 주어집니다.

정수를 직접 숫자형으로 변환하지 않고,  
각 자리 숫자의 합을 이용해 9로 나눈 나머지를 구하는 문제입니다.

어떤 수를 9로 나눈 나머지는  
그 수의 각 자리 숫자의 합을 9로 나눈 나머지와 같습니다.

## 제한 사항

- `1 ≤ number의 길이 ≤ 100,000`
- `number`는 숫자로만 이루어져 있습니다.
- `number`는 정수 0이 아니라면 숫자 `'0'`으로 시작하지 않습니다.

## 입출력 예

| number | result |
|---|---|
| "123" | 6 |
| "78720646226947352489" | 2 |

## 문제 풀이

`number`의 길이가 최대 100,000이므로 `int`, `long` 같은 숫자형으로 변환하면 범위를 초과할 수 있습니다.

따라서 문자열을 한 글자씩 순회하면서 각 자리 숫자를 더한 뒤, 마지막에 9로 나눈 나머지를 반환합니다.

문자 숫자는 그대로 더하면 문자 코드값이 더해지므로,  
`num - '0'`을 사용해 실제 숫자로 변환해야 합니다.

### 사용된 메서드 정리

- `toCharArray()`
  - 문자열을 문자 배열로 변환합니다.

- `num - '0'`
  - 문자 숫자를 정수 숫자로 변환합니다.
  - 예: `'7' - '0' == 7`

- `%`
  - 나머지 연산자입니다.

## 풀이 코드

```java
class Solution {
    public int solution(String number) {
        int answer = 0;

        for (char num : number.toCharArray()) {
            answer += num - '0';
        }

        return answer % 9;
    }
}
