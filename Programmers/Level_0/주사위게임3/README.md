# 주사위 게임 3

## 문제 설명

1부터 6까지 숫자가 적힌 주사위 4개를 굴렸을 때, 나온 숫자의 조합에 따라 점수를 계산하는 문제입니다.

점수 규칙은 다음과 같습니다.

- 네 숫자가 모두 같으면 `1111 × p`
- 세 숫자가 같고 나머지 하나가 다르면 `(10 × p + q)^2`
- 두 개씩 같은 값이면 `(p + q) × |p - q|`
- 한 쌍만 같으면 나머지 두 수의 곱 `q × r`
- 모두 다르면 가장 작은 수

## 제한 사항

- `a, b, c, d`는 1 이상 6 이하의 정수입니다.

## 입출력 예

| a | b | c | d | result |
|---|---|---|---|--------|
| 2 | 2 | 2 | 2 | 2222 |
| 4 | 1 | 4 | 4 | 1681 |
| 6 | 3 | 3 | 6 | 27 |
| 2 | 5 | 2 | 6 | 30 |
| 6 | 4 | 2 | 5 | 2 |

## 문제 풀이

이 문제는 네 개의 주사위 값을 비교해서 경우를 나누는 문제입니다.

직접 `a == b`, `b == c`처럼 모든 경우를 비교할 수도 있지만, 경우의 수가 많아져 코드가 복잡해질 수 있습니다.

따라서 주사위 값을 배열에 넣고 정렬한 뒤, 정렬된 배열의 패턴을 기준으로 점수를 계산했습니다.

예를 들어 `[4, 1, 4, 4]`는 정렬하면 `[1, 4, 4, 4]`가 됩니다.  
이 경우 `arr[1] == arr[3]`이므로 세 개의 값이 같은 경우로 판단할 수 있습니다.

사용한 주요 메서드는 다음과 같습니다.

- `Arrays.sort(arr)`
  - 배열을 오름차순으로 정렬합니다.
  - 정렬하면 같은 숫자들이 서로 붙기 때문에 경우를 쉽게 구분할 수 있습니다.

- `Math.pow(a, b)`
  - 거듭제곱을 계산합니다.
  - 반환 타입이 `double`이므로, 정수로 사용하려면 `(int)` 형변환이 필요합니다.

- `Math.abs(value)`
  - 절댓값을 구합니다.
  - 두 수의 차이를 양수로 계산할 때 사용했습니다.

## 풀이 코드

```java
import java.util.Arrays;

class Solution {
    public int solution(int a, int b, int c, int d) {
        int[] arr = {a, b, c, d};
        Arrays.sort(arr);

        // 4개가 모두 같은 경우
        if (arr[0] == arr[3]) {
            return 1111 * arr[0];
        }

        // 3개가 같은 경우
        if (arr[0] == arr[2] || arr[1] == arr[3]) {
            int p = arr[1];
            int q = (arr[0] == arr[2]) ? arr[3] : arr[0];

            return (int) Math.pow(10 * p + q, 2);
        }

        // 2개씩 같은 경우
        if (arr[0] == arr[1] && arr[2] == arr[3]) {
            int p = arr[0];
            int q = arr[2];

            return (p + q) * Math.abs(p - q);
        }

        // 한 쌍만 같은 경우
        if (arr[0] == arr[1]) {
            return arr[2] * arr[3];
        }

        if (arr[1] == arr[2]) {
            return arr[0] * arr[3];
        }

        if (arr[2] == arr[3]) {
            return arr[0] * arr[1];
        }

        // 모두 다른 경우
        return arr[0];
    }
}
