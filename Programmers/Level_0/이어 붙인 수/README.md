# 이어 붙인 수

## 문제 설명

정수가 담긴 리스트 `num_list`가 주어집니다. `num_list`의 홀수만 순서대로 이어 붙인 수와 짝수만 순서대로 이어 붙인 수의 합을 return 하도록 `solution` 함수를 완성해주세요.

## 제한사항

- `2 ≤ num_list`의 길이 `≤ 10`
- `1 ≤ num_list`의 원소 `≤ 9`
- `num_list`에는 적어도 한 개씩의 짝수와 홀수가 있습니다.

## 입출력 예

| num_list | result |
|---|---:|
| `[3, 4, 5, 2, 1]` | 393 |
| `[5, 7, 8, 3]` | 581 |

## 입출력 예 설명

### 입출력 예 #1

홀수만 이어 붙인 수는 `351`이고 짝수만 이어 붙인 수는 `42`입니다. 두 수의 합은 `393`입니다.

### 입출력 예 #2

홀수만 이어 붙인 수는 `573`이고 짝수만 이어 붙인 수는 `8`입니다. 두 수의 합은 `581`입니다.

## 문제 풀이

이 문제는 배열에 들어 있는 숫자들을 순서대로 확인하면서, 홀수는 홀수끼리 이어 붙이고 짝수는 짝수끼리 이어 붙인 뒤 두 수를 더하면 되는 문제입니다.

먼저 문자열 `odd`와 `even`을 준비합니다.

- 현재 숫자가 홀수이면 `odd` 뒤에 이어 붙입니다.
- 현재 숫자가 짝수이면 `even` 뒤에 이어 붙입니다.

이렇게 하면 예를 들어 `[3, 4, 5, 2, 1]`의 경우,

- 홀수 문자열: `"351"`
- 짝수 문자열: `"42"`

가 만들어집니다.

그 다음 `Integer.parseInt()`를 사용해 문자열을 정수로 바꾼 뒤, 두 값을 더해서 반환하면 됩니다.

현재 코드는 이 과정을 그대로 구현한 정답 코드입니다.

## 풀이 코드

```java
class Solution {
    public int solution(int[] num_list) {
        String odd = "";
        String even = "";
        
        for (int num : num_list) {
            if (num % 2 == 0) {
                even += num;
            } else {
                odd += num;
            }
        }
        
        return Integer.parseInt(even) + Integer.parseInt(odd);
    }
}
```
