# 조건 문자열

## 문제 설명

문자열에 따라 다음과 같이 두 수의 크기를 비교하려고 합니다.

두 수가 `n`과 `m`이라면 다음과 같습니다.

- `">", "="` : `n >= m`
- `"<", "="` : `n <= m`
- `">", "!"` : `n > m`
- `"<", "!"` : `n < m`

두 문자열 `ineq`와 `eq`가 주어집니다.  
`ineq`는 `<`와 `>` 중 하나이고, `eq`는 `=`와 `!` 중 하나입니다.  
그리고 두 정수 `n`과 `m`이 주어질 때, `n`과 `m`이 `ineq`와 `eq`의 조건에 맞으면 `1`, 아니면 `0`을 return하도록 `solution` 함수를 완성해주세요.

### 제한 사항

- `1 ≤ n, m ≤ 100`

### 입출력 예

| ineq | eq | n  | m  | result |
|------|----|----|----|--------|
| `<`  | `=` | 20 | 50 | 1 |
| `>`  | `!` | 41 | 78 | 0 |

### 입출력 예 설명

- 입출력 예 #1  
  `20 <= 50`은 참이기 때문에 `1`을 return합니다.

- 입출력 예 #2  
  `41 > 78`은 거짓이기 때문에 `0`을 return합니다.

---

## 문제 풀이

이 문제는 문자열 `ineq`와 `eq`의 조합에 따라 `n`과 `m`을 어떤 방식으로 비교할지 결정하는 문제입니다.

먼저, 문자열 비교에는 `==`가 아니라 `equals()` 메소드를 사용해야 합니다.  
Java에서 `==`는 문자열의 내용을 비교하는 것이 아니라 주소를 비교하기 때문입니다.  
따라서 `ineq.equals(">")`, `eq.equals("=")`처럼 작성해야 정확하게 문자열 값을 비교할 수 있습니다.

또한 두 조건을 동시에 확인하기 위해 논리 연산자 `&&`를 사용했습니다.  
예를 들어 `ineq.equals(">") && eq.equals("=")`는 `ineq`가 `">"`이고 동시에 `eq`가 `"="`인 경우를 의미합니다.

각 조건에 따라 비교식은 다음과 같이 나뉩니다.

- `">", "="` 이면 `n >= m`
- `"<", "="` 이면 `n <= m`
- `">", "!"` 이면 `n > m`
- `"<", "!"` 이면 `n < m`

이 비교 결과는 `boolean answer` 변수에 저장합니다.  
`boolean`은 참 또는 거짓을 저장하는 자료형이므로, 비교 결과를 담기에 적절합니다.

마지막에는 삼항 연산자를 사용해 결과를 반환합니다.

```java
return answer ? 1 : 0;
```

## 풀이 코드

```java
class Solution {
    public int solution(String ineq, String eq, int n, int m) {
        boolean answer = false;
        
        if (ineq.equals(">") && eq.equals("="))
            answer = n >= m;
        else if (ineq.equals("<") && eq.equals("="))
            answer = n <= m;
        else if (ineq.equals(">") && eq.equals("!"))
            answer = n > m;
        else
            answer = n < m;          
        return answer ? 1 : 0;
    }
}
```
