# 수열과 구간 쿼리 2

## 문제 설명

정수 배열 `arr`와 2차원 정수 배열 `queries`가 주어집니다. `queries`의 원소는 각각 하나의 query를 나타내며, `[s, e, k]` 꼴입니다.

각 query마다 순서대로 `s ≤ i ≤ e`인 모든 `i`에 대해 `k`보다 크면서 가장 작은 `arr[i]`를 찾습니다.

각 쿼리의 순서에 맞게 답을 저장한 배열을 반환하는 `solution` 함수를 완성해 주세요.  
단, 특정 쿼리의 답이 존재하지 않으면 `-1`을 저장합니다.

## 제한사항

- `1 ≤ arr`의 길이 `≤ 1,000`
- `0 ≤ arr`의 원소 `≤ 1,000,000`
- `1 ≤ queries`의 길이 `≤ 1,000`
- `0 ≤ s ≤ e < arr`의 길이
- `0 ≤ k ≤ 1,000,000`

## 입출력 예

| arr | queries | result |
|---|---|---|
| `[0, 1, 2, 4, 3]` | `[[0, 4, 2], [0, 3, 2], [0, 2, 2]]` | `[3, 4, -1]` |

## 입출력 예 설명

### 입출력 예 #1

첫 번째 쿼리의 범위에는 `0, 1, 2, 4, 3`이 있으며 이 중 `2`보다 크면서 가장 작은 값은 `3`입니다.

두 번째 쿼리의 범위에는 `0, 1, 2, 4`가 있으며 이 중 `2`보다 크면서 가장 작은 값은 `4`입니다.

세 번째 쿼리의 범위에는 `0, 1, 2`가 있으며 여기에는 `2`보다 큰 값이 없습니다.

따라서 `[3, 4, -1]`을 return 합니다.

## 문제 풀이

이 문제는 각 쿼리마다 주어진 구간 `[s, e]`를 직접 확인하면서, `k`보다 큰 값들 중 가장 작은 값을 찾으면 되는 문제입니다.

먼저 정답을 저장할 배열 `answer`를 만들고, 기본값을 모두 `-1`로 채워 둡니다.

```java
Arrays.fill(answer, -1);
```

이렇게 해두면 해당 쿼리에서 조건을 만족하는 값이 하나도 없을 때 자동으로 `-1`이 유지됩니다.

이후 `queries`를 하나씩 꺼내서 다음 정보를 구합니다.

- `s`: 탐색 시작 인덱스
- `e`: 탐색 끝 인덱스
- `k`: 기준값

각 쿼리마다 `s`부터 `e`까지 반복하면서 `arr[i] > k`인 값을 찾습니다.  
조건을 만족하는 값을 처음 발견했으면 그 값을 바로 저장하고, 이미 저장된 값이 있다면 `Math.min()`을 사용해 더 작은 값을 유지합니다.

즉, 한 쿼리 안에서는 다음 과정을 반복합니다.

- `arr[i] > k`이면 후보가 될 수 있음
- 아직 값이 정해지지 않았으면 그 값을 저장
- 이미 후보가 있다면 둘 중 더 작은 값으로 갱신

이 과정을 모든 쿼리에 대해 반복하면 각 쿼리의 답을 순서대로 구할 수 있습니다.

## 풀이 코드

```java
import java.util.Arrays;

class Solution {
    public int[] solution(int[] arr, int[][] queries) {
        int[] answer = new int[queries.length];
        Arrays.fill(answer, -1);
        
        int idx = 0;
        for (int[] rule : queries) {
            int s = rule[0], e = rule[1], k = rule[2];
            
            for (int i = s; i <= e; i++) {
                if (arr[i] > k) {
                    answer[idx] = answer[idx] == -1 ? arr[i] : Math.min(answer[idx], arr[i]);
                }
            }
            
            if (answer[idx] == 1000001) {
                answer[idx] = -1;
            }
            
            idx++;
        }
        
        return answer;
    }
}
```
