# 주사위 게임 2

## 문제 설명

1부터 6까지 숫자가 적힌 주사위가 세 개 있습니다. 세 주사위를 굴렸을 때 나온 숫자를 각각 `a`, `b`, `c`라고 했을 때 얻는 점수는 다음과 같습니다.

- 세 숫자가 모두 다르다면 `a + b + c` 점을 얻습니다.
- 세 숫자 중 어느 두 숫자는 같고 나머지 다른 숫자는 다르다면 `(a + b + c) × (a² + b² + c²)` 점을 얻습니다.
- 세 숫자가 모두 같다면 `(a + b + c) × (a² + b² + c²) × (a³ + b³ + c³)` 점을 얻습니다.

세 정수 `a`, `b`, `c`가 매개변수로 주어질 때, 얻는 점수를 return 하는 `solution` 함수를 작성해 주세요.

## 제한사항

- `a`, `b`, `c`는 1 이상 6 이하의 정수입니다.

## 입출력 예

| a | b | c | result |
|---|---|---|---:|
| 2 | 6 | 1 | 9 |
| 5 | 3 | 3 | 473 |
| 4 | 4 | 4 | 110592 |

## 입출력 예 설명

### 입출력 예 #1

예제 1번에서 세 주사위 숫자가 모두 다르므로 `2 + 6 + 1 = 9`점을 얻습니다. 따라서 `9`를 return 합니다.

### 입출력 예 #2

예제 2번에서 두 주사위 숫자만 같으므로 `(5 + 3 + 3) × (5² + 3² + 3²) = 11 × 43 = 473`점을 얻습니다. 따라서 `473`을 return 합니다.

### 입출력 예 #3

예제 3번에서 세 주사위 숫자가 모두 같으므로 `(4 + 4 + 4) × (4² + 4² + 4²) × (4³ + 4³ + 4³) = 12 × 48 × 192 = 110592`점을 얻습니다. 따라서 `110592`를 return 합니다.

## 문제 풀이

이 문제는 세 수 `a`, `b`, `c`가 **모두 같은지**, **두 개만 같은지**, **모두 다른지**를 구분해서 조건에 맞는 식을 계산하면 됩니다.

경우는 총 세 가지입니다.

- `a == b && b == c` 이면 세 수가 모두 같은 경우
- `a == b || b == c || c == a` 이면 두 수만 같은 경우
- 그 외에는 모두 다른 경우

먼저 세 수가 모두 같은 경우를 가장 먼저 검사해야 합니다.  
만약 이 조건을 먼저 확인하지 않으면, 세 수가 모두 같은 경우도 `a == b || b == c || c == a` 조건에 포함되어 버리기 때문입니다.

각 경우에 맞는 점수 계산식은 다음과 같습니다.

- 모두 다름: `a + b + c`
- 두 수만 같음: `(a + b + c) × (a² + b² + c²)`
- 모두 같음: `(a + b + c) × (a² + b² + c²) × (a³ + b³ + c³)`

코드에서는 `Math.pow()`를 사용해 제곱과 세제곱을 계산했고, `Math.pow()`의 반환형이 `double`이므로 `int`로 형변환해서 사용했습니다.

## 풀이 코드

```java
class Solution {
    public int solution(int a, int b, int c) {
        if (a == b && b == c) {
            return (a + b + c)
                    * (int)(Math.pow(a, 2) + Math.pow(b, 2) + Math.pow(c, 2))
                    * (int)(Math.pow(a, 3) + Math.pow(b, 3) + Math.pow(c, 3));
        } else if (a == b || b == c || c == a) {
            return (a + b + c)
                    * (int)(Math.pow(a, 2) + Math.pow(b, 2) + Math.pow(c, 2));
        } else {
            return a + b + c;
        }
    }
}
```
