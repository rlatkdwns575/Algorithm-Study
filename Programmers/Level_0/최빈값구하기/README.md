# 최빈값 구하기

## 문제 설명

최빈값은 주어진 값 중에서 가장 자주 나오는 값을 의미합니다. 정수 배열 `array`가 매개변수로 주어질 때, 최빈값을 return 하도록 `solution` 함수를 완성해보세요. 최빈값이 여러 개면 `-1`을 return 합니다.

## 제한사항

- `0 < array`의 길이 `< 100`
- `0 ≤ array`의 원소 `< 1000`

## 입출력 예

| array | result |
|---|---:|
| `[1, 2, 3, 3, 3, 4]` | 3 |
| `[1, 1, 2, 2]` | -1 |
| `[1]` | 1 |

## 입출력 예 설명

### 입출력 예 #1

`[1, 2, 3, 3, 3, 4]`에서 `1`은 1개, `2`는 1개, `3`은 3개, `4`는 1개로 최빈값은 `3`입니다.

### 입출력 예 #2

`[1, 1, 2, 2]`에서 `1`은 2개, `2`는 2개로 최빈값이 `1`, `2`입니다. 최빈값이 여러 개이므로 `-1`을 return 합니다.

### 입출력 예 #3

`[1]`에는 `1`만 있으므로 최빈값은 `1`입니다.

## 문제 풀이

이 문제는 배열에 각 숫자가 몇 번 나왔는지 세고, 그중 가장 많이 등장한 값을 찾는 문제입니다.

배열의 원소 범위가 `0 이상 999 이하`이므로, 각 숫자의 등장 횟수를 저장할 수 있는 크기 `1000`의 배열을 만들어 빈도를 기록할 수 있습니다.

```java
int[] index = new int[1000];
```

이후 입력 배열 `array`를 순회하면서 해당 숫자의 등장 횟수를 1씩 증가시킵니다.

```java
index[array[i]]++;
```

그러면 `index[x]`에는 숫자 `x`가 몇 번 등장했는지가 저장됩니다.

그 다음 빈도 배열을 0부터 999까지 확인하면서 가장 많이 등장한 횟수를 `max_num`에 저장하고, 그때의 숫자를 `answer`에 저장합니다.

- 현재 숫자의 빈도가 `max_num`보다 크면  
  새로운 최빈값이므로 `max_num`과 `answer`를 모두 갱신합니다.
- 현재 숫자의 빈도가 `max_num`과 같으면  
  최빈값이 여러 개라는 뜻이므로 `answer`를 `-1`로 바꿉니다.

이 과정을 끝까지 수행하면,

- 최빈값이 하나뿐이면 그 값을 반환
- 최빈값이 여러 개면 `-1` 반환

하게 됩니다.

현재 코드는 이 방법을 그대로 구현한 정답 코드입니다.

## 풀이 코드

```java
class Solution {
    public int solution(int[] array) {
        int[] index = new int[1000];
        
        for (int i = 0; i < array.length; i++) {
            index[array[i]]++;
        }
        
        int max_num = 0;
        int answer = 0;
        
        for (int i = 0; i < 1000; i++) {
            if (index[i] > max_num) {
                max_num = index[i];
                answer = i;
            } else if (index[i] == max_num) {
                answer = -1;
            }
        }
        
        return answer;
    }
}
```
